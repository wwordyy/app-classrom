
## Требования

Перед запуском убедитесь, что у вас установлены:

-   Node.js (рекомендуется 18+)
-   npm
-   PostgreSQL
-   Git

------------------------------------------------------------------------

# Установка проекта

## 1. Клонировать репозиторий

``` bash
git clone https://github.com/wwordyy/app-classrom.git
cd app-classrom
```

------------------------------------------------------------------------

# Настройка Backend

## 2. Создать файл окружения

Перейдите в папку `backend` и создайте файл `.env`.

Путь:

    backend/.env

Содержимое файла:

``` env
PORT=3001
DATABASE_URL="postgresql://yourLogin:yourPassword@localhost:5432/db_app_classroom"
REMEMBER_JWT_SECRET="yourCode"
```

------------------------------------------------------------------------

## 3. Создать папки для загрузок

В корне `backend` создайте папку `uploads` со следующей структурой:

    backend
     ├ uploads
     │  ├ avatars
     │  ├ merged
     │  └ posts

Команда для создания:

``` bash
mkdir -p uploads/avatars uploads/merged uploads/posts
```

------------------------------------------------------------------------

## 4. Установить зависимости backend

``` bash
cd backend
npm install
```

------------------------------------------------------------------------

## 5. Создать базу данных

Перед выполнением миграций создайте базу данных в PostgreSQL:

    db_app_classroom

------------------------------------------------------------------------

## 6. Выполнить миграции Prisma

``` bash
npx prisma migrate dev --name init
```

------------------------------------------------------------------------

## 7. Заполнить базу тестовыми данными

``` bash
npx prisma db seed
```

------------------------------------------------------------------------

## 8. Запустить backend

``` bash
node server.js
```

Backend будет доступен по адресу:

    http://localhost:3001

------------------------------------------------------------------------

# Настройка Frontend

Откройте новый терминал.

## 9. Установить зависимости

``` bash
cd frontend
npm install
```

------------------------------------------------------------------------

## 10. Запустить frontend

``` bash
npm run dev
```

------------------------------------------------------------------------

# Открыть приложение

После запуска frontend откройте в браузере:

    http://localhost:5173/
